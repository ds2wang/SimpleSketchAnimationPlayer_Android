Android app

Views:

Main view: default view, allows you to choose background color, frame rate and file

Background color view: after selecting a color and clicking done, it goes back to the main view

File choose view: will go back to main view after selecting a valid xml file. It will then go to animation view

Animation view: upon loading, will play the animation
pause by tapping, play by tapping again
to exit animation view, double tap while animation is still playing
to reset to beginning, double tap when animation is finished
